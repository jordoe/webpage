Angular 2+

Proyecto realizado en Angular 8, sobre la API PokeApi. La aplicación consiste en una "pokédex" que permite buscar pokémons y consultar sus detalles.

Se realizan llamadas asíncronas a la API, se utiliza el routing de Angular y está organizada por componentes. Además, está traducida a varios idiomas mediante ngx-translate.

Es una app responsive: está adaptada para pantallas de PC, de móvil y de tablet.

Ionic

Aplicación realizada con Ionic 4 y Angular 8, sobre la API TheMovieDb.

Consiste en un buscador de películas con dos tipos de búsqueda: un módulo que devuelve una película aleatoria basada en los filtros introducidos, y otro en el que se permite encontrar películas mediante búsqueda de texto, ordenándolos como prefiera el usuario.

También pueden marcarse las películas como "guardada", "vista" y "favorita". Así mismo, pueden consultarse detalles tanto de las películas como de los directores y actores.

Tiene traduciones con ngx-translate y, además de ser responsive al igual que la app anterior, saca provecho de Ionic para mostrar elementos nativos, como los "picker", de forma distinta dependiendo del tipo de dispositivo (Android o iOs).
